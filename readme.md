Establishes a peer to peer connection via peerjs. Simply enter the peer's id
and await their acceptance. Messages will be stored in the browser's local
storage for later reference.
##### Setup
1. `cp env .env`
2. `yarn`
3. `yarn build`
4. `yarn start`
